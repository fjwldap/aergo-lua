# Aergo JDBC

### JDBC

: java에서 DB 프로그래밍을 할때 사용하는 api

<br/>
### Aergo JDBC : 블록체인에서 DB 쉽게 조작

계약 + JDBC 패키지로 구성

lua 계약을 Aergo 블록체인에 배포하면 DB계약이 블록체인의 일부가 된다

-> DB계약(=address)

sql을 사용하면 transaction, query 호출

<br/>

### lua 프로그래밍 언어 사용

- db.exec : 테이블 생성/삭제

        db.exec([[create table ~~]])
        db.exec("insert into ~~")

- db.query : 테이블 조회

        db.query("select ~~")

- db.prepare : placeholder을 가진 sql문 생성..
